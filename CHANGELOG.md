# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-28
### Added
- Decorative background "mesh glows" for enhanced depth.
- `CHANGELOG.md` and `LICENSE` files.
- Global entrance animations (`.reveal`) for components.

### Changed
- **UI Modernization**: Updated the entire design system to a premium "Cyber-Purple" aesthetic.
- **Typography**: Switched primary font to **Outfit** (Google Fonts).
- **Navigation**: Redesigned top nav with glowing accent line and interactive search bar.
- **Sidebar**: Improved translucency and glassmorphism effects.
- **Tokens**: Refined color palette with deeper darks and more vibrant accents.
- **Responsive Layout**: Improved spacing and container widths for modern displays.

### Fixed
- Improved accessibility for theme toggle button.
- Refined table cell padding and contrast.

## [1.0.0] - 2026-02-10
### Added
- Initial release of BoutiqueFlow.
- Financial management module.
- Admin and Super Admin dashboards.
- Customer showroom and cart functionality.
- Cloudinary integration for product images.
- MongoDB Atlas backend support.
